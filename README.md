# OPS/ICD German clinical code system explorer

This dash application allows to visualize ICD/OPS codes as network.
Code almost entirely generated by Claude Sonnet.

![AI Usage](https://img.shields.io/badge/👤_█████████░_🤖-90%25_AI-e879f9?style=flat-square)


Features:
- search query on labels
- code pattern seach query
- code information pannel with year range validity

The app reads the data from a parquet file that has been generated from online available .xml files containing the codes metdata. 
The DataFrame contains the following columns: "code", "label", "parent", "kind",  "is_leaf", "year", "catalogue".
Currently, the data covers the year range 2013-2026.

## Usage

- install uv
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
- install dependencies 
```bash
uv sync
```
- launch dash app
```bash
uv run python app.py
```
